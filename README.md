## How to Use

1. Click on numbers to input values
2. Click on an operator (+, -, *, /)
3. Enter the second number
4. Click "=" to see the result
5. Click "Clear" to reset

## Error Handling

- **Division by Zero**: Displays "Error: Cannot divide by zero!" message
- **Invalid Input**: Prevents multiple decimal points in a number

- **Conditionals**: If-else statements for operation selection
